"Έξυπνος" Διακόπτης

Με την συζήτηση γύρω από την ενέργεια να βρίσκεται τον τελευταίο καιρό στο αποκορύφωμά της, η ομάδα των Roboholics σκέφτηκε να δημιουργήσει τον "έξυπνο" διακόπτη φωτός. Ο διακόπτης θα περιέχει έναν αισθητήρα φωτός, ο οποίος θα ελέγχει αν υπάρχει αρκετός φωτισμός στην αίθουσα διδασκαλίας του σχολείου. Αν αυτό συμβαίνει, ο διακόπτης θα ενεργοποιείται και θα σβήνει τα φώτα, ώστε να γίνεται εξοικονόμηση ενέργειας. Επίσης, ο διακόπτης θα μπορεί να συνδεθεί με το ίντερνετ, ώστε να είναι δυνατός ο απομακρυσμένος έλεγχος του. Έτσι, σε περίπτωση που έχει ξεχαστεί ανοιχτό το φως σε κάποια από τις αίθουσες, θα υπάρχει η δυνατότητα να σβήσει με ένα κλικ από το κινητό μας.
Είναι να μην είμαστε χαρούμενοι εμείς κι "έξυπνος" αυτός; 
Για να το νοστιμίσουμε κιόλας, έξυπνος στα αρχαία ελληνικά ήταν αυτός που δεν ήταν...κοιμήσης! Άγρυπνος, λοιπόν, ο διακόπτης μας...

Υλικά που θα χρειαστούμε
•	Ένα μικροελεγκτή με ασύρματη διασύνδεση (Arduino η ESP)
•	Αισθητήρες φωτός
•	Σερβοκινητήρες 
•	3d εκτυπωτή
•	Νήμα εκτύπωσης
